# AI Powered Resume Based Job Recommendation System

A Streamlit web application that:
- Extracts resume data from PDF
- Summarizes skills using Gemini AI
- Identifies skill gaps
- Suggests a future roadmap
- Recommends jobs from LinkedIn & Naukri

## Tech Stack
- Python
- Streamlit
- Google Gemini API

## Setup (Local)
```bash
pip install -r requirements.txt
streamlit run app.py
